About Me

웹퍼블리싱 분야를 깊이 있게 다져가고자 하는 웹퍼블리셔 정이슬입니다.
디자인 된 이미지를 웹 브라우저에 구현하는 것에 큰 재미를 느껴 웹퍼블리싱 공부를 시작했으며, 보다 전문성을 갖추고 싶은 욕심이 생겨 디자이너에서 퍼블리셔로의 전향에 도전하게 되었습니다. 기본이 탄탄한 것이 가장 중요하다는 생각으로 웹표준과 웹접근성을 준수하고, 유지보수가 편리한 코딩을 위해 늘 고민하고 있습니다.
항상 배우고 탐구하는 자세로 발전해나가는 웹퍼블리셔가 되겠습니다.

What i handle Publishing Skills

  • HTML5

    웹 표준과 웹 접근성을 준수하는 HTML 정보 구조설계로 웹사이트 퍼블리싱을 해오고 있습니다. 특히 웹 표준을 위해 HTML5에서 권장하는 마크업 방식과 시멘틱 태그를 사용해 전체 레이아웃을 설계합니다. 웹사이트를 퍼블리싱 하기 전에는 ‘기능 정의서'를 만들고 기능정의서에 전체적인 폰트 서식, 컬러 가이드, 사이즈 정보, 상호작용 여부, HTML 와이어프레임까지 꼼꼼하게 미리 정리해 놓고 체계적인 퍼블리싱 작업을 수행합니다.

  • CSS3

    웹 표준을 준수하는 CSS 코딩으로 크로스브라우징 (Cross Browsing)을 위해 안정적인 CSS 속성을 사용해서 HTML Element를 배치하고 디자인합니다. Internet Explorer를 배제하는 경우 CSS3 속성을 적극적으로 활용합니다. CSS 선택자의 코딩 순서는 HTML 순서에 따라 순서에 맞게 코딩하고 적절한 주석을 사용합니다.

  • SCSS(SASS)

    현재 CSS가 제공하는 것보다 훨씬 나은 워크 플로우를 위해서 SCSS(SASS)도 익혀가고 있습니다. 아직 SCSS(SASS)를 활용한 충분한 프로젝트 경험은 없지만, 개인 포트폴리오 홈페이지에 SCSS를 사용한 퍼블리싱 결과물이 있습니다. SCSS로 프로젝트를 진행하는 능력이 필요한 경우, 빠르게 적응하여 생산성을 낼 수 있도록 계속해서 공부하고 있습니다.

  • Jquery/Javascript

    네비게이션, 탭 메뉴, 어코디언, 모달창 등의 기능은 플러그인을 사용하지 않고 직접 하드 코딩으로 제작합니다. 자바스크립트를 공부하는 데에 흥미를 가지고 있으며 현재 역량이 개발자 수준이 되지는 못하지만 부족한 점을 채우기 위해 계속해서 노력하고 있습니다.

  • Responsible Web

    반응형 웹 제작에 있어서 크로스브라우징(Cross Browsing)을 위해 안정적인 방식인 float, inlinb-block, position, Media Query를 활용한 반응형 웹 제작 경험을 충분히 가지고 있으며, flex와 gird 방식으로도 반응형 웹 사이트 제작이 가능합니다.

  • GitHub

    인프런에서 GitHub 입문 강의를 수강하였습니다. 강의에서 협업 시뮬레이션을 통해 간접적으로나마 Github를 통한 협업을 경험했습니다. 실제로 팀원들과 협업을 경험해본 것은 아니라 부족한 점이 많겠지만 기초적인 개념은 익혔다고 생각합니다. 이를 바탕으로 회사에서 Github를 통한 협업을 필요로 할 경우 빠르게 익히고 적응하도록 노력하겠습니다.

What i handle Design Skills

  • Photoshop / Illustrator

    웹사이트 UI 디자인 작업은 포토샵을 주로 사용하며, 웹 UI 디자인 작업을 위한 포토샵 활용은 능숙하게 사용 가능합니다. 또한 이벤트 디자인, 배너 디자인을 위한 포토샵과 일러스트 활용이 가능합니다.

  • Figma

    Figma를 이용한 웹 UI 디자인 작업이 가능합니다. UI 작업에 특화 된 Figma의 세세한 기능들은 조금씩 익혀가고 있습니다.
